                Embed Image into Page
                < img> is used to embed image into page.
                Basic Attributes:
                1. alt : It specifies the alternative text to display when
                image fails to load.
                < img alt="some message">
                2. src : It specifies the virtual path of image.
                < img src="images/mobile.jpg">
                3. title : It specifies the title to display as screen tip.
                < img src="" alt="" title="some description">
                4. width & height : It defines the width and height of
                image in pixels or % Size in % will make image fluid.
                < img width="50%" height="200">
                5. border : It define border for image.
                < img border="1">
                6. align : It defines image alignment left or right.
                [image will not have center align]
                It allows to wrap text around image.
                < img align="left">
                7. vspace & hspace: Horizontal and Vertical padding
                space.
                < img hspace="10" vspace="30">
                Advanced Attributes:
                8. crossorigin : It defines access restriction for image,
                when it is accessed over network.
                a) use-credentials
                b) anonymous
                < img src="from any api" crossorigin="anonymous">
                9. decoding : It defines the loading style
                a) sync
                b) async
                sync will block relative content while loading image.
                async will load along with other content in page.
                < img decoding="async">
                10. importance : It sets priority for image
                a) low
                b) high
                c) auto
                < img importance="high|low|auto">
