Unordered List:
                It is similar to ordered list, but contains a symbol instead
                of number.
                It supports "type=circle, disc, square"
                Syntax:-
                
    
                
  •  Item-1 
    • 
                
  •  Item-2 
    • 
                

                FAQ: How to set custom symbol?
                Ans : By using "CSS" list-style attribute.
                Syntax:-
                ul {
                list-style-image: url("images/bullet.png");
                }
                FAQ: Can we define different symbol for every item in list?
                Ans: No.
                Symbols from 3rd Party:
                    1. Bootstrap Icons
                    2. Fontawesome Icons
                    3. Material Icons
                    4. Telerik Icons
                    Setup Bootstrap Icons for Website:
                    1. Download and Install Bootstrap icons library for project
                    > npm install bootstrap-icons --save [from terminal]
                    https://icons.getbootstrap.com/
                    2. Link "bootstrap-icons.css" to your page
                    < link rel="stylesheet" href="../node_modules/bootstrapicons/font/bootstrap-icons.css">
                    3. Set icon by using Class Name
                    < span class="bi bi-cart"> < /span>
                    < li class="bi bi-house"> < /li>
