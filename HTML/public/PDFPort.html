Projects

Insurance Report Generation System

Final Semester Project for Government

Spring Boot JSP MySQL Bootstrap Data JPA

This Java Spring Boot-based Insurance Report Generation System manages and processes citizens' insurance records with various plans (Cash, Food, Medical, Employment) and statuses (Approved, Denied, Terminated).

The application dynamically loads plan names and statuses from the database and supports advanced search functionalities. Users can filter records based on criteria, with an option to view all records if no filter is selected.

The system also provides export options, allowing users to download records as Excel or PDF files, which are automatically emailed as attachments.

  • Environment: JDK17, Spring Boot, Data JPA, MySQL, JSP, Bootstrap
