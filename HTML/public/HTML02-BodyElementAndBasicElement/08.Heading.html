Headings need a different appearance in page. -HTML provides pre-defined heading levels. -HTML headings levels are defined by using . -“n” refers to level number from 1 to 6. Syntax: < h1> < h2> ................... < h6> -Heading levels from 1 to 6 will decrease the size. -Heading can be defined with “align” attribute, which can align left, center, right or justify. FAQ: Why to use heading element for headings? -Heading elements will make your topics SEO friendly. -SEO can identify the topics on your page if they are defined in heading. Note: -Don’t use headings for highlighting any word or sentence in a paragraph. -Heading by default will have line break above and below. -Don’t use too many headings in a page. SEO can SPAM your page.
